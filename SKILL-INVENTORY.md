**AI/ML**: `claude-api`, `langchain-langgraph-agents`, `pydantic-ai`, `rag-engineer`, `vector-database-engineer`

### 사용법
```bash
# 어느 프로젝트에서나
skill-install                          # 대화형 (fzf)
skill-install --skills "스킬명" --dest project-agents   # 비대화형
```

프로젝트에 skill-manager 설치 후 Codex에서 `$skill-manager install` 또는 `skill-manager 실행해줘`를 입력하면 AI가 자동 추천·설치합니다.
