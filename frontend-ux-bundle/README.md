# 🛸 Antigravity Frontend & UX Bundle (Expert Edition)

**안티 그래비티(Antigravity) 및 AI 코딩 에이전트의 디자인 지능과 프런트엔드 숙련도를 비약적으로 높여주는 정예 스킬 번들입니다.** 

이 번들은 현대적인 제품 개발(SaaS, 랜딩 페이지, 대시보드 등)에 필요한 **프런트엔드 아키텍처**와 **제품 디자인 철학**의 정수만을 엄선하여 재구성했습니다. Next.js, Tailwind CSS, Shadcn UI 환경에서 프리미엄급 결과물을 도출하는 가장 완벽한 방법입니다.

---

## 🧠 1. Core Intelligence (에이전트 제어 및 워크플로우)
에이전트(저)의 '사고하는 방식'을 결정하며, 복잡한 요구사항을 스스로 분해하고 실행하게 합니다.

- **`antigravity-meta-intelligence`**: 당면한 과제에 가장 적합한 스킬을 스스로 선택하고, 목표 달성을 위한 최적의 경로를 탐색하는 지휘관 역할을 합니다.
- **`antigravity-fullstack-workflow`**: SaaS MVP 기획부터 데이터베이스 설계, API, 프런트엔드 구현에 이르는 전체 개발 사이클을 체계적으로 관리합니다.
- **`antigravity-premium-design`**: 안티 그래비티만의 미래지향적이고 감각적인 스타일(Glassmorphism, Vibrant Colors, Modern Dark Mode)을 모든 산출물에 기본 적용합니다.

## 🎨 2. Design & UX Strategy (기획 및 시각 설계)
사용자가 감탄할 수 있는 시각적 완성도와 매끄러운 사용성을 보장합니다.

- **`ux-design-intelligence`**: JTBD(Jobs-to-be-Done) 프레임워크와 유저 페르소나 분석을 통해 근거 있는 사용자 경험을 설계합니다.
- **`apple-standard-product-design`**: 애플의 HIG(Human Interface Guidelines)를 기반으로 하여, 정교한 여백(Spacing), 타이포그래피, 아이콘 배치를 통해 '명품'의 느낌을 줍니다.
- **`ux-audit / flow / copy`**: 사용자의 이탈을 막는 검수 프로세스, 자연스러운 화면 전환 설계, 그리고 사용자의 행동을 유도하는 매력적인 문구(Microcopy)를 작성합니다.
- **`ui-design-checker`**: 구현된 UI가 디자인 시스템과 일치하는지, 접근성(WCAG 2.2)이나 시각적 일관성에 결함이 없는지 픽셀 단위로 검수합니다.

## 💻 3. Frontend Engineering (기술 스택 및 구현)
가장 빠르고 견고하며 유지보수가 쉬운 코드를 작성하도록 가이드합니다.

- **`react-nextjs-development`**: Next.js App Router, Server Components, Partial Prerendering 등 최신 성능 최적화 기법을 완벽히 이해하고 적용합니다.
- **`typescript-expert`**: 런타임 에러를 최소화하는 고난도의 타입 시스템을 구축하며, 대규모 협업에 적합한 추상화 아키텍처를 제안합니다.
- **`tanstack-query-expert`**: 비동기 데이터 페칭, 캐싱, 낙관적 업데이트(Optimistic Update) 등 복잡한 서버 데이터 동기화를 전문적으로 처리합니다.
- **`tailwind-design-system`**: 단순한 스타일 적용을 넘어, 디자인 토큰과 테마 시스템이 결합된 지속 가능한 스타일 구조를 설계합니다.
- **`motion-animation-optimizer`**: Framer Motion을 사용하여 사용자 경험을 해치지 않으면서도 생동감 넘치는 마이크로 Interaction을 고성능으로 구현합니다.

## 🛠️ 설치 및 사용법 (Installation)

이 번들을 전역(Global)에 설치하면, 어떤 프로젝트 폴더에서든 안티 그래비티가 이 강력한 기준들을 바탕으로 협업하게 됩니다.

```bash
# 1. 레포지토리 클론
git clone https://github.com/[유저네임]/antigravity-frontend-ux-bundle.git
cd antigravity-frontend-ux-bundle

# 2. 전역 설치 (Global Install)
antigravity install .
```

## ✨ 기대 효과
- **최상위 디자인 퀄리티**: 픽셀 완벽주의와 현대적 미학이 결합된 고품질 UI/UX 산출물 도출.
- **아키텍처 레벨의 코딩**: 단순 기능 구현을 넘어 확장성 있고 견고한 프런트엔드 코드베이스 구축.
- **의사결정 가이드**: 에이전트가 디자인과 기술 선택의 기로에서 최적의 해답을 스스로 제안.
