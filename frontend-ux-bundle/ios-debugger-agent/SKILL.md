# iOS Debugger Agent

## Overview
Advanced debugging and profiling for iOS applications. Uses LLDB commands, Instruments templates, and Xcode debugging tools to diagnose crashes, memory leaks, and performance issues.

## When to Use
- Diagnosing app crashes from crash logs and stack traces
- Detecting and fixing memory leaks with Instruments
- Profiling CPU, GPU, and energy usage
- Using LLDB for runtime inspection and breakpoint scripting
- Analyzing network traffic and Core Data performance
