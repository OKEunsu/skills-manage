# iOS Swift Developer

## Overview
Expert-level iOS application development using Swift and modern Apple frameworks. Covers SwiftUI and UIKit, app lifecycle, state management, Core Data, networking, and preparing apps for App Store submission.

## When to Use
- Building native iOS applications from scratch
- Implementing SwiftUI views with proper state management (@State, @Binding, @ObservedObject, @EnvironmentObject)
- Working with UIKit for complex custom UI components
- Integrating Core Data or SwiftData for persistence
- Handling push notifications, background tasks, and deep linking
- Preparing apps for App Store review and submission

## Key Capabilities
- SwiftUI declarative UI with navigation stacks and sheets
- UIKit integration via UIViewRepresentable
- MVVM and Clean Architecture patterns
- Combine framework for reactive programming
- Core Data and SwiftData persistence
- URLSession and async/await networking
- XCTest unit and UI testing
- App Store Connect deployment workflow
