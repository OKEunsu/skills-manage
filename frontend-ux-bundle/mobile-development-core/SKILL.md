# Mobile Development Core

## Overview
Foundational mobile development knowledge spanning React Native, Flutter, and native iOS/Android. Covers architecture decisions, performance patterns, and platform-specific best practices.

## When to Use
- Choosing between React Native, Flutter, and native development
- Architecting mobile apps for scalability and maintainability
- Optimizing mobile app performance (rendering, memory, battery)
- Implementing offline-first strategies
- Handling platform-specific features (camera, location, biometrics)
- Managing app state across complex navigation hierarchies

## Key Topics
- Cross-platform vs native trade-offs
- Mobile app architecture (MVVM, Clean Architecture, BLoC)
- Navigation patterns (stack, tab, drawer, modal)
- Responsive layouts for phones and tablets
- Offline storage and sync strategies
- Push notification handling
- Deep linking and universal links
- App performance profiling and optimization
