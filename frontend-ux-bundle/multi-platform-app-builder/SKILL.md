# Multi-Platform App Builder

## Overview
Design and build applications targeting multiple platforms (iOS, Android, Web, Desktop) from a shared codebase. Covers framework selection, shared business logic, platform-specific adaptations, and deployment strategies.

## When to Use
- Building apps that must run on both iOS and Android
- Extending mobile apps to web or desktop
- Sharing business logic across platforms
- Implementing platform-adaptive UI patterns

## Key Frameworks
- React Native with Expo for iOS/Android/Web
- Flutter for iOS/Android/Web/Desktop
- Kotlin Multiplatform for shared business logic
- Capacitor for web-to-mobile wrapping
