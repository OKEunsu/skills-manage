# Unity Game Developer

## Overview
Build games and interactive experiences using the Unity engine and C#. Covers game architecture, physics, animation, UI, and deploying to mobile, web, and desktop platforms.

## When to Use
- Building 2D or 3D games with Unity
- Implementing game mechanics, physics, and AI
- Creating responsive game UI with Unity's UI Toolkit or UGUI
- Optimizing performance for mobile devices
- Setting up multi-platform build pipelines

## Key Topics
- MonoBehaviour lifecycle and component architecture
- Physics (Rigidbody, Colliders, Raycasting)
- Animation controllers and state machines
- Scriptable Objects for data-driven design
- Addressables and asset bundle management
- Platform-specific build settings and optimization
