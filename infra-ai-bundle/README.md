# 🛸 CODEX & Infra Master Combined (120+ Skills)

**안티 그래비티(Antigravity) 및 AI 코딩 에이전트의 '인프라 설계, 백엔드 아키텍처, AI 엔지니어링' 지능을 집대성한 통합 엔진입니다.**

이 폴더는 120개가 넘는 스킬들을 단일 평면(Flat) 구조로 관리하여, 에이전트가 어떤 복잡한 기술적 요구사항도 지연 없이 즉각적으로 찾아내어 실행할 수 있도록 최적화되었습니다.

## 🏗️ 주요 핵심 스킬 분야 (Main Capabilities)

### 1. 🤖 Next-Gen AI Engineering
최신 언어 모델 및 에이전트 프레임워크를 자유자재로 다룹니다.
- **`gemini-api-dev` / `claude-api`**: 글로벌 최상위 LLM API 최적화 및 연동.
- **`hugging-face-cli` / `embedding-strategies`**: 오픈소스 모델 활용 및 벡터 임베딩 전략 수립.
- **`crewai` / `computer-use-agents`**: 자율적으로 동작하는 멀티 에이전트 시스템 및 컴퓨터 제어 에이전트 구축.

### ☁️ 2. Cloud & DevOps Infrastructure
전 세계 어디서든 서비스가 안정적으로 돌아가는 기반을 설계합니다.
- **`aws-skills` / `gcp-cloud-run` / `cloudflare-workers`**: 다중 클라우드 환경 최적화 및 서벌리스 아키텍처.
- **`docker-expert` / `helm-chart-scaffolding` / `gitops-workflow`**: 컨테이너 기반 배포 및 자동화된 인프라 관리.
- **`cost-optimization` / `aws-cost-cleanup`**: 클라우드 비용을 최소화하고 자원 효율을 극대화하는 운영 지능.

### ⚙️ 3. Advanced Backend & Data Architecture
견고하고 확장 가능한 데이터 시스템을 구축합니다.
- **`graphql-architecture` / `grpc-go`**: 효율적인 API 통신 시스템 설계.
- **`cqrs` / `event-sourcing`**: 고가용성을 보장하는 최신 이벤트 기반 데이터 아키텍처.
- **`clickhouse` / `convex-platform`**: 정교한 실시간 데이터 분석 및 현대적인 데이터 플랫폼 활용.

## 🛠️ 설치 및 사용법 (Installation)

이 레포지토리를 전역에 설치하면, 당신의 AI 에이전트가 단순한 코더를 넘어 **"클라우드 설계자이자 AI 시스템 엔지니어"**의 관점에서 문제를 해결하기 시작합니다.

```bash
# 1. 레포지토리 클론
git clone https://github.com/[유저네임]/antigravity-infra-ai-bundle.git
cd antigravity-infra-ai-bundle

# 2. 전역 설치 (Global Install)
antigravity install .
```
