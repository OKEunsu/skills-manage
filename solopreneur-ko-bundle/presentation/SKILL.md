# Presentation Skill

발표 목적에 맞는 슬라이드 구조와 콘텐츠를 설계한다.

## 슬라이드 구성 원칙

- 슬라이드당 핵심 메시지 1개
- 텍스트 최소화, 시각화 우선
- 스토리텔링: 문제 → 해결 → 증거 → 행동 유도

## 지원 형식

- **Notion** — 노션 페이지 슬라이드 구조
- **Canva/Figma** — 슬라이드 콘텐츠 + 레이아웃 가이드
- **Marp (Markdown)** — 마크다운 기반 슬라이드 (코드 전달 시)
- **PowerPoint** — 슬라이드 구조 및 내용 (실제 파일 생성은 Codex slides 스킬 활용)

## 출력 형식

슬라이드별:
```
[슬라이드 N] 제목
- 핵심 메시지: ...
- 포함할 내용: ...
- 시각화 제안: ...
```

- 전체 스크립트 (발표자 노트)
- 디자인 톤 & 컬러 제안

## 행동 규칙

- 청중과 목적 먼저 확인 (투자자/팀내/고객 발표 여부)
- 분량 기준 제시 (5분=10슬라이드, 20분=20슬라이드)
- 한국어 기본, 영어 발표 요청 시 영어로
